# 🚀 Portfólio Dev FrontEnd | Diogo Borges

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi construído para apresentar meus projetos, certificações e habilidades de forma moderna, limpa e com alta performance.

## 🎨 Design & Estilo
A aplicação foi completamente projetada em torno do conceito de **Glassmorphism**, com elementos semitransparentes (efeito vidro) flutuando sobre um fundo escuro imersivo com efeitos sutis de luz (glowing orbs). O formato do site é **One-Page**, ou seja, todas as seções (Home, Sobre e Projetos) ficam em uma única página interligada por rolagens suaves.

## 🛠️ Tecnologias Utilizadas

- **[React 18](https://reactjs.org/)**: Biblioteca base do projeto.
- **[TypeScript](https://www.typescriptlang.org/)**: Adiciona tipagem estática que previne erros no desenvolvimento.
- **[Vite](https://vitejs.dev/)**: Ferramenta de build que torna o desenvolvimento e recarregamento super-rápidos.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para garantir o visual moderno e 100% responsivo.
- **[Ant Design Icons](https://ant.design/components/icon/)**: Biblioteca de ícones utilizada nos botões e no footer.

## ✨ Funcionalidades Principais

- **Scroll Suave e Ativo**: Barra de navegação que marca dinamicamente qual seção o usuário está lendo no momento (usando `IntersectionObserver`).
- **Animações de Entrada (ScrollReveal)**: Cada projeto e elemento de texto entra na tela de forma suave (staggered fade-in) à medida que se rola a página para baixo.
- **Tema de Código ("Editor View")**: A seção *Sobre Mim* imita a interface clássica do Visual Studio Code / Terminal, dando um ar bem autêntico de desenvolvimento.
- **Slider Vertical**: Carrossel flexível que percorre as certificações.
- **Totalmente Responsivo**: Layout estruturado com CSS Grid e Flexbox que se adapta nativamente para celulares, tablets e desktops grandes.

## 🚀 Como rodar o projeto localmente

### 1. Pré-requisitos
Tenha instalado em sua máquina o [Node.js](https://nodejs.org/).

### 2. Passos para a instalação

Clone este repositório para o seu computador:
```bash
git clone https://github.com/diogoborges4/portifolio-vite.git
```

Entre na pasta do projeto:
```bash
cd portifolio-vite
```

Instale as dependências usando NPM (ou Yarn/Pnpm):
```bash
npm install
```

Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

Acesse no seu navegador, geralmente em `http://localhost:5173`.
